# signature/board-nested — Ed25519 golden signature

**TEST KEY — DO NOT USE IN PRODUCTION.** This keypair exists only to pin a
deterministic golden signature for cross-implementation conformance.

- `seed.hex` — 32-byte test seed (0x42 repeated).
- `public.hex` — public key derived from the seed using Go `crypto/ed25519`.
- `message.canonical` — the exact bytes signed: the canonical form of the
  `canonical/board-nested` document.
- `signature.hex` — Ed25519 signature (deterministic for a given key+message).

Generated by `go run ./tools/w3a gen-vectors`.
