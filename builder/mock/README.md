# W3A Mock Provider Demo

The builder v1 demo uses `mock://` backends only. A mock demo is generated by:

```sh
go run ./tools/w3a build --answers ../builder/examples/board.answers.json --out /tmp/board.draft.w3a --mock-demo /tmp/board.mock-demo.json
```

The generated JSON is an inspectable runtime-demo fixture: it lists actions,
their risk class, whether confirmation is enforced, and sample in-memory data.
It is not a real provider integration and never stores credentials.
